C4C Sending Client Console

COMMANDS:
a to send play samples on all active clients,
x to stop playing samples on all active clients,
q to send test tones,
1, 2,3, 4, 5 to change send modes (0 is test, 1 is Seth, 2 is Lainie, 3 is Mario),
6, 7, 8, 9, 0 to change frequencies,
n to cancel update timer

Send trigger to all clients to start current sample at time in textbox
